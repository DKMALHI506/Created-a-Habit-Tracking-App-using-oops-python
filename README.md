# Habit Tracking App using Python (OOP)

This is a simple Habit Tracking Application built using
Python and Object-Oriented Programming (OOP) concepts.

The app allows users to:
- Add daily habits
- Mark habits as completed
- Store habit data using JSON

## Concepts Used
- Python Classes and Objects
- File Handling (JSON)
- Date and Time Module
- Menu-driven program

## Project Structure
- habit.py – Habit class
- main.py – Main program logic
- habits.json – Stores habit data
